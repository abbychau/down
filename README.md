# Oekaki Chat

インターン課題用作成した「お絵かきチャット」の汎用モジュールです（多分）　node.js+socket.ioで動きます。


### 使い方
#### インストール
#####1.モジュールをインストール
	cd server
	npm install
	npm

#####2.アプリを起動
	cd server
	node app.js

#####2.クライアントを起動
`/client-sample/index.html`をブラウザで開く
